# ecsdeployer secrets get

Get the value of a specific secret

```
ecsdeployer secrets get VARIABLE_NAME [VARIABLE_NAME...] [flags]
```

## Options

```
      --bare            Only return the value. You can only provide a single VARIABLE_NAME when using this.
  -c, --config file     Configuration file to check
  -f, --format string   The output format to use (dotenv, plain) (default "dotenv")
  -h, --help            help for get
```

## Global Options

```
      --debug   Enable debug mode
```
