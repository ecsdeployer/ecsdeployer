# ecsdeployer jsonschema

outputs ECS Deployer's JSON schema

```
ecsdeployer jsonschema [flags]
```

## Options

```
  -h, --help            help for jsonschema
  -o, --output string   where to save the json schema (default "-")
```

## Global Options

```
      --debug   Enable debug mode
```
