# ecsdeployer secrets list

List secrets in SSM that will be added to this project

```
ecsdeployer secrets list [flags]
```

## Options

```
  -c, --config file     Configuration file to check
  -f, --format string   The output format to use (dotenv, plain) (default "dotenv")
  -h, --help            help for list
```

## Global Options

```
      --debug   Enable debug mode
```
